Fonctionnement git :

Tout d'abord, on clone un projet existant avec :
'git clone (url du projet git)'

Ensuite on travaille dans le dossier respectif, puis quand on veut push sur git, on crée un commit, c'est comme remplir le colis que tu vas envoyer a la poste :
'git commit -a' <- -a veut dire tout ce qui est dans le dossier, comme ca pas de jaloux

Enfin pour push, c'est a dire déposer le colis au bureau de poste pour l'envoi, on fait :
'git push -u origin (branche souhaitée)' <- -u veut dire [ce que dit la doc en gros #jspdutoutmdrr](https://git-scm.com/docs/git-push/fr#git-push---set-upstream)
